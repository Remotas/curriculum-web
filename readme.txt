📌 Descripción del Proyecto

Este repositorio contiene el CV digital profesional de Melquiales Farias, desarrollado con tecnologías web modernas para garantizar:
✅ Diseño limpio y profesional (réplica exacta del formato PDF original)
✅ Totalmente responsivo (adaptable a móviles, tablets y desktop)
✅ Versión bilingüe (español/inglés) con cambio de idioma integrado
✅ Optimizado para impresión (modo PDF-friendly)
✅ Interacciones mejoradas (efectos hover, enlaces activos)

🚀 Instrucciones de Uso

1. Visualización
Versión en español: Abrir index.html en cualquier navegador web.

Versión en inglés: Abrir index_en.html.

2. Funcionalidades clave
🔹 Cambio de idioma: Botón superior derecho (*English/Español*).
🔹 Descarga PDF: Botón para guardar el CV en formato PDF.
🔹 Enlaces interactivos:

📞 Teléfono: Llama directamente al hacer clic.

✉️ Email: Abre el cliente de correo predeterminado.

🔗 LinkedIn: Redirecciona al perfil en nueva pestaña.

📍 Ubicación: Abre Google Maps con la dirección.

3. Personalización
Para adaptar este CV a otro perfil:

Reemplazar profile.png con una imagen cuadrada (recomendado: 500×500 px).

Modificar el contenido en los archivos HTML (texto, fechas, enlaces).

Actualizar el archivo PDF en /assets/pdf/.

Ajustar colores editando las variables CSS en :root.
